# Más Ferretería - Sitio Web (test)

Sitio web oficial de Más Ferretería, ubicada en Ushuaia, Tierra del Fuego, Argentina.


## 🚀 Despliegue

### Render

1. Conecta tu repositorio a Render
2. Crea un nuevo "Static Site"
3. Usa la configuración de `render.yaml`
4. El sitio se desplegará automáticamente

### Local

```bash
# Iniciar servidor local
python3 -m http.server 8000

# O usar npm
npm start
```

## 🛠️ Tecnologías

- **HTML5** Semántico
- **Tailwind CSS** para estilos
- **JavaScript Vanilla** para interactividad
- **Google Maps API** para ubicación
- **Font Awesome** para iconos
