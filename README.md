## 🚀 Despliegue

### Render

1. Conecta tu repositorio a Render
2. Crea un nuevo "Static Site"
3. Usa la configuración de `render.yaml`
4. El sitio se desplegará automáticamente

### Local

```bash
# Iniciar servidor local
python3 -m http.server 8000

# O usar npm
npm start
```
